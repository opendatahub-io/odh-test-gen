# Scope Boundaries

This skill does NOT:

- **Create the initial test plan** — use `/test-plan-create` for that
- **Create the PR** — use `/test-plan-publish` for that
- **Re-run sub-agent analyzers** — if feedback requires deeper re-analysis with new source documents, use `/test-plan-update`
- **Change the PR title, description, or reviewers**
- **Blindly apply all feedback** — every item is assessed and presented to the user for decision
